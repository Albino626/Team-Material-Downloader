### 📋 技术栈

- **前端**: HTML + CSS + JavaScript
- **后端**: Node.js + Express
- **浏览器自动化**: Puppeteer
- **HTTP客户端**: Axios
- **跨域支持**: CORS

### 🔧 配置项

- **默认端口**: 3000（可配置）
- **默认密码**: 321123（可修改）
- **监听地址**: 0.0.0.0（支持远程）
- **清理间隔**: 12小时（可配置）

### 📊 系统要求

- **Node.js**: >= 14.0.0
- **操作系统**: Windows / macOS / Linux
- **内存**: 建议 4GB+
- **磁盘**: 建议 10GB+（包含Puppeteer）
